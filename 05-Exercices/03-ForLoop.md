## For loop

### 1 Double
```javascript
const arr = [1, 2, 3];
double(arr);

// [2, 4, 6];
```

### 2 Sum
```javascript
const arr = [1, 2, 3];
sum(arr);

// 6
```

### 3 SomeEven
```javascript
const arr = [1, 2, 3];
someEven(arr);

// true
```

### 4 EveryOdd
```javascript
const arr = [1, 2, 3];
everyOdd(arr);

// false
```

### 5 Max value
```javascript
const arr = [1, 2, 3];
max(arr);

// 3
```

### 6 Find index value in array
```javascript
const arr = [1, 2, 3];
findIndex(1, arr);
// 0

findIndex(7, arr);
// -1
```
