## Types

### Primitive types
* Number `42`
* String `'abc'`
* Boolean `true`
* Function `function() {}` /!\ arg and return

### Complex types
* Array `[1, 2, 3]`
* Object `{ foo: 'bar }`

### Examples

`const str = 'hello world';`

`let int = 12;`

`const arr = [1, 2, 3];  arr[1]`;

`const obj = { foo: 'bar' }; foo.bar`;
